Was ist HTML?

HTML (Hyper Text Markup Lanuage) ist eine Beschreibungs- oder Auszeichnungssprache zum Erstellen von Webseiten.

Wie erstelle ich eine Webseite?

Es stehen verschiedene Möglichkeiten zur Erstellung zur Verfügung:

  • Texteditoren (Windows Editor, Notepad++, usw.)
  • HTML-Editoren
    1. Textgestüzt (Visual Studio Code, ...)
    2. WYSIWYG (Dreamweaver, ...)
